The Challenge: Transforming Raw Data into Intelligence
Problem Statement
The raw hotel booking dataset, containing over 119,000 reservations, was unstructured, incomplete, and contained quality issues. The core challenge was to:
  • Perform **extensive data cleaning** to ensure reliability.
  • Conduct **Exploratory Data Analysis (EDA)** to identify key trends.
  • Translate complex data patterns into **clear, actionable business recommendations**.
Data Issues Identified
  • Missing header row, requiring external research for column names.
  • Significant missing values in key columns (`country`, `agent`, `company`).
  • Invalid records (e.g., bookings with zero guests).
  • Inconsistent data types requiring careful conversion.
