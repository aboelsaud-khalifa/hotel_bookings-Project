## Overview
Analyzing hotel booking data to understand cancellation patterns and customer trends.

## Files
* **hotel_bookings_analysis.ipynb**: Python code for cleaning and EDA.

## Key Insights
* High cancellation rate (approx 37%).
* August is the busiest month.
* "Online TA" is the main booking channel.
